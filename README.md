# BetterTrims Tweaks

Mixin addon for [BetterTrims](https://modrinth.com/mod/bettertrims) 2.3.2.

## What this does

Fixes prismarine shard trim not making guardians ignore you. Elder guardians still attack (mining fatigue immunity is handled by BetterTrims itself).
