# BetterTrims Tweaks

**BetterTrims Tweaks** is a Fabric mod mixin for [BetterTrims](https://modrinth.com/mod/bettertrims) 2.3.2 that provides additional fixes and gameplay adjustments.

## Features

### Prismarine Shard Fix
- **Guardian Visibility**: Fixes the Prismarine Shard trim effect to properly prevent Guardians from targeting players.
  - Requires a configurable number of Prismarine trim pieces.
  - Elder Guardians are unaffected and will still target the player (as intended).
